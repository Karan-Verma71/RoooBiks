# rubiks-cube-solver

This is a Algorithmic C++, OOPS , and Modelling based project which has some optimal 
algorithms to solve a scrambled rubiks cube.

It uses some commonly known Graph Algorithms like DFS , BFS and Some Non-Trivial algorithms like IDDFS.

Apart From the above algorithms the project uses a research paper published by Korf on how to solve rubiks cube optimally
by using the IDA* Algorithm has also been separately implemented in this project

[Korf's Research Paper](https://www.cs.princeton.edu/courses/archive/fall06/cos402/papers/korfrubik.pdf)

It uses a prebuild Pattern Database text file which it uses to do the Heuristic Algorithm of Iterative Deepening A (IDA**) algorithm
to find the optimal number of moves to reduce the effective search depth rather than the branching factor, making it significantly more efficient than brute-force methods .
